    R Is For Actuaries

    I have recorded a 4 part video series that introduces actuaries to R. The course walks you through an experience study using the following:

    • Data cleaning with dplyr.
    • Data visualization with ggplot2
    • Creating a report with markdown
    • Publishing results using Shiny

    I published the final dashboard and markdown document using shinyapps.io. The dashboard also contains links to the video tutorials and source data.

    SOA Predictive Analytics Summary


    In 2019 I completed the SOA Predictive Analytics Seminar. My notes are summarized at the link below. Some things to point out:


    • The notes are written in Markdown which allows you to combine narrative notes, code snippets, and visualizations.
    • Normally I use R Studio to code in R and R Markdown. For these notes I used Jupyter Notebook - which renders better when pushed to Git Hub.
    • These notes are a great resource for actuaries. It is difficult to find Predictive modeling notes that are designed with actuarial work in mind.
